# BobECS

Entity Component System library

### Installing
Get module:
```
npm install --save bobecs
```

### Exemple
See [exemples](exemples)

## Running the tests

Unit tests:
```
npm run tu
```

CS tests:
```
npm run cs
```

All tests:
```
npm run test
```
