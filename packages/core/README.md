# BobECS

Entity Component System library

[![Build Status](https://travis-ci.com/desaintvincent/bobecs.svg?branch=master)](https://travis-ci.com/desaintvincent/bobecs)


### Installing
Get module:
```
npm install --save bobecs
```

### Exemple
See [exemples](exemple)

## Running the tests

Unit tests:
```
npm run tu
```

CS tests:
```
npm run cs
```

All tests:
```
npm run test
```
